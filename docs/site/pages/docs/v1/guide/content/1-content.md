### Guide

# Content Authoring (v1)

## Legacy content format

This documents the v1 content format. For the current approach, see [Content Authoring](page:content).

### Basic Markdown

V1 supported standard Markdown with limited frontmatter options.

### Limitations

- No tagged code blocks
- Limited frontmatter parameters
- No semantic parsing

V2 introduces richer content capabilities with semantic parsing and tagged code blocks.
