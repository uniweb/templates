# Installation

## Step-by-step installation guide

Follow these steps to install and set up your project.

### Using pnpm (recommended)

The fastest way to get started is using pnpm:

- Open your terminal
- Run the create command
- Follow the prompts

### Using npm

You can also use npm if you prefer:

- Run npx to execute the create command
- Choose your template
- Install dependencies

### Manual installation

For advanced users who want more control:

- Clone the repository
- Install dependencies manually
- Configure your environment

After installing, proceed to [Configuration](page:configuration) to set up your project.
