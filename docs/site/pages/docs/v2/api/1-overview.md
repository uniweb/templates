### API Reference

# Overview

## Complete API documentation

This section provides detailed API documentation for all available functions, components, and configuration options.

Use this reference when you need specific details about how things work under the hood.

### Organization

The API reference is organized by category:

- Configuration options
- Component props
- Utility functions
- CLI commands

### Conventions

Throughout this documentation, we use the following conventions:

- Required parameters are marked with an asterisk
- Optional parameters show their default values
- Types are indicated in parentheses

See also the [Getting Started](page:getting-started) guide for an introduction to key concepts.
